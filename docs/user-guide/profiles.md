# Profiles

Profiles allow you to quickly apply a predefined set of configurations suitable for different use cases.

## Available Profiles

*   **Beginner:** Basic setup with essential tools and maximum guidance.
*   **Intermediate:** Balanced setup for standard development needs.
*   **Advanced:** minimal setup with maximum control for experienced users.
*   **Minimal:** Only the bare necessities (system + security).
*   **Full:** Everything enabled.

## Using a Profile

You can select a profile during the wizard or via CLI:

```bash
vps-configurator install --profile intermediate
```
