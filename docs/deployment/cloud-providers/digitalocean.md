# Deploying on DigitalOcean

## Droplet Creation

1. Create a Droplet.
2. Select **Debian 12** image.
3. Choose plan (Basic - Regular - 2GB RAM recommended).
4. Add your SSH Key.

## Post-Launch

1. SSH into your droplet:

    ```bash
    ssh root@your-droplet-ip
    ```

2. Run the Quick Install script.
